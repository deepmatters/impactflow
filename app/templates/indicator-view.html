Indicator bank update instruction

If you as an admin want to update the indicator bank, please follow these steps:

  1. Prepare an Excel or a CSV file with 1 sheet. Label the columns of the 1st row (From A1, B1, and so on) as category, subcategory, indicator_en, indicator_th, source_en, source_th respectively. Note all column names do not use capital letters.
  2. From Row 2 enter indicator information with respect to the corresponding column name.
  3. Save and send the file to chitpong [at] changefusion [dot] org. We will replace the indicator data in the database with the new one.
